📌 LANGKAH YANG HARUS DILAKUKAN:

1. **Gunakan API CoinMarketCap untuk ambil harga spot**
   - Endpoint: `https://pro-api.coinmarketcap.com/v1/cryptocurrency/quotes/latest`
   - Header: `'X-CMC_PRO_API_KEY': os.getenv("COINMARKETCAP_API_KEY")`
   - Ambil harga berdasarkan `symbol` dari user input `/analyze BTC`

2. **Hitung Support dan Resistance otomatis**
   - Support = Harga sekarang × 0.97 (turun 3%)
   - Resistance = Harga sekarang × 1.03 (naik 3%)

3. **Tambahkan deteksi Area Supply & Demand (SnD)** berbasis logika sederhana:
   - Area Demand = harga sekarang - 3.5% s.d. -2.5%
   - Area Supply = harga sekarang + 2.5% s.d. +3.5%
   - Contoh jika harga $29,000:
     - Demand: $27,985 - $28,275
     - Supply: $29,725 - $30,015

4. **Perbaiki format Telegram agar tidak error:**
   - Gunakan `parse_mode="HTML"` dan pastikan:
     - Tidak ada simbol `<` atau `>` di teks biasa
     - Tidak ada tag HTML seperti `<b>` tanpa penutup
     - Hindari karakter `_`, `*`, `[`, `]` jika pakai `Markdown`

5. **Tambahkan error handler:**
   - Jika gagal ambil data dari API, tampilkan:
     ❌ Gagal mengambil data analisis untuk BTC
