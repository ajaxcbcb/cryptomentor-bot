Diagnosa dan selesaikan masalah koneksi ke CoinGlass API versi v4.

✅ Langkah-langkah yang harus dilakukan Agent:
1. Coba akses endpoint: https://open-api-v4.coinglass.com/public/v1/futures/tickers
2. Gunakan API Key yang tersimpan di secrets: COINGLASS_API_KEY
3. Tambahkan header X-API-Key dengan nilai dari secret tersebut
4. Cetak status code dan hasil JSON-nya
5. Periksa apakah data yang keluar adalah dummy (ciri: harga/static, timestamp tidak berubah)
6. Jika dummy → reset API Key secara manual atau siapkan template pesan untuk support CoinGlass
7. Jika respons sukses dan real-time → tampilkan sukses ✅

📩 Jika dummy → buat pesan untuk dikirim ke Telegram support CoinGlass:
https://t.me/coinglass

Format pesan:
---
Halo tim CoinGlass,
Saya sudah upgrade ke STARTUP plan dan mendapatkan API Key: <API_KEY>.
Namun saat saya mengakses endpoint API v4, hasilnya masih dummy.
Saya sudah pakai header X-API-Key, reset API, dan mengganti endpoint ke v4.
Mohon bantuannya untuk aktivasi penuh akun saya. Terima kasih.
---

💡 Tambahan:
Jika tetap dummy, bantu arahkan ke solusi alternatif seperti Binance API untuk futures/open interest secara real-time.
"""

import os
import requests

api_key = os.getenv("COINGLASS_API_KEY")
url = "https://open-api-v4.coinglass.com/public/v1/futures/tickers"
headers = {"X-API-Key": api_key}

print("🚀 Mengakses API CoinGlass...")

res = requests.get(url, headers=headers)
print("Status Code:", res.status_code)

try:
    data = res.json()
    if res.status_code != 200:
        print("❌ Gagal mengakses API.")
    elif "data" in data and isinstance(data["data"], list):
        sample = data["data"][0]
        print("✅ Respons Data Ditemukan:")
        print(sample)

        # Cek apakah dummy (misalnya harga tetap, atau timestamp aneh)
        if sample.get("price", 0) in [0, 1, 1000] or sample.get("symbol") == "DUMMY":
            print("⚠ Kemungkinan besar ini DUMMY DATA.")
            print("\n📩 Siapkan pesan ke CoinGlass Support (Telegram):")
            print(f"""
Halo tim CoinGlass,
Saya sudah upgrade ke STARTUP plan dan mendapatkan API Key: {api_key}
Namun saat saya mengakses endpoint API v4, hasilnya masih dummy.
Saya sudah pakai header X-API-Key, reset API, dan mengganti endpoint ke v4.
Mohon bantuannya untuk aktivasi penuh akun saya. Terima kasih.
""")
        else:
            print("✅ Data tampaknya valid & real-time.")
    else:
        print("⚠ Data tidak ditemukan.")
except Exception as e:
    print("❌ Error parsing JSON:", e)
